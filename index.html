Tipos de Extintores

Conoce los distintos tipos de extintores: Agua, CO₂, Polvo químico seco, Espuma, entre otros. Elige el adecuado para tu empresa y protección.

Extintor de Agua

Ideal para incendios de clase A (materiales sólidos como madera y papel).

Extintor de CO₂

Usado para incendios de clase B (líquidos inflamables) y clase C (equipos eléctricos).

Extintor de Polvo Químico

Versátil para incendios de clase A, B y C. Es muy efectivo para apagar fuegos de diferentes tipos.

Extintor de Espuma

Ideal para incendios de líquidos inflamables (clase B) y sólidos (clase A) que generan mucha radiación de calor.
